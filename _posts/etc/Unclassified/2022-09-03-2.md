# 협업?
## 제품 제작 과정
1. Wireframe
    - 어떤 제품을 만들지 정리하는 과정
    - 기능과 정보가 담긴 구조
2. design
    - 제품의 디자인(사용하는 색상, 아이콘, 위젯 디자인등)

1. Task 작성
    - Task 분리
    - Task 분담
2. 공동 작업을 위한 규칙 작성
    - Naming Convention
    - Git Branch Convention
    - commit 단위
    - PR 단위
    - 기타
3. 진행 상황 공유
    - 스크럼, 회고 등
4. 코드리뷰
5. 테스트 (QA)
6. 머지
